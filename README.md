# Clash of Clans Base Collector Bot

## Overview

This project is a Python-based bot that collects Clash of Clans bases by scraping the web and posts them to a Discord channel along with a picture of the base. It provides a convenient way to share Clash of Clans bases with your community on Discord.

## Features

- **Web Scraping:** The bot uses web scraping techniques to collect Clash of Clans bases from various sources on the web.
- **Discord Integration:** Bases are posted to a designated Discord channel, providing a centralized location for sharing and discussing strategies.
- **Base Visualization:** Each base is accompanied by a picture for easy viewing and analysis.

## Requirements

- Python 3.x
- Discord.py library
- Requests library
